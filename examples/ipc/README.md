# IPC Examples

Run the server and client scripts in separate terminals.

## Simple RPC

```bash
kgpy simple_server.kg
```

```bash
kgpy simple_client.kg
```

## Broadcast

```bash
kgpy srv_broadcast.kg
```

```bash
kgpy cli_broadcast.kg
```

## Pub/Sub

```bash
kgpy srv_pubsub.kg
```

```bash
kgpy cli_pubsub.kg
```
